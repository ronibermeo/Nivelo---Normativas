# Nivelo — Normativas

Reglamento interno unificado de **Nivelo** en formato HTML: documento único con estilos integrados, tabla de contenidos y lectura cómoda en pantalla.

## Contenido

- `index.html` — Reglamento interno unificado (abrir en el navegador).

## Uso local

Abre `index.html` con tu navegador (doble clic o arrastrar el archivo a una ventana del navegador). No requiere servidor para lectura básica.

## GitHub Pages (opcional)

En el repositorio: **Settings → Pages → Source**: rama `main`, carpeta `/ (root)`. La URL será similar a `https://ronibermeo.github.io/Nivelo---Normativas/`.
